Projecting coordinates across spatial reference systems made easy.

I've been immersed in GIS applications over the past few months, one thing I found interesting were how so few free, easy to use coordinate conversion tools existed. If you buy an expensive GIS server you usually get the bells and whistles of a geometric service that includes projection functionality, but I've whipped together a quick converter that is free to use.

Check it out here

Here's a snippit of the javascript code I used to rip values from the page and POST to a php file


$(document).ready(function() {
  // When id with Action is clicked
  $('#PostJSON').click(function() {
  
  	var data = "mercatorX_lon=" + $('#mercator_x').val() + "&mercatorY_lat=" + $('#mercator_y').val()
  			 + "&inputsr=" + $('#inputsr').val() + "&outputsr=" + $('#outputsr').val();           

             $.ajax({
		  url: "/yourphpfilenamehere.php",
		  data: data,
		  type: "POST",
		  success: function(data, textStatus, jqXHR){
		  var data = JSON.parse(data);
		  console.log();
			$("#converted_to_geo").text("X: " + data.geometries[0].x + ", Y: " + data.geometries[0].y);
		  },
		  error: function (jqXHR, textStatus, errorThrown){
			console.log('Error ' + jqXHR);
		  }
		});

  });
});
